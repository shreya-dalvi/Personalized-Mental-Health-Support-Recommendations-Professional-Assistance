📌 Overview
MindMend is a comprehensive mental health platform that assesses
depression severity using MCQ responses, audio recordings, and video
submissions. Based on these inputs, the platform predicts depression
severity and provides personalized mental health resources, AI
assistance, motivational content, and professional support.
The project was developed as a final-year B.E. Computer Engineering
capstone at AISSMS Institute of Information Technology, Pune, under
Savitribai Phule Pune University.
